Antilatency device network

Библиотека для комуникации с устройствами Antilatency. Позволяет находить устройства среди подключенных, отслеживать подключение новых устройств, запускать задачи на устройствах, а также обновлять прошивку.

Инициализация 

						using System;
Для начала работы с `Antilatency device network` нужно загрузить библиотеку и создать экземпляр `ILibrary`
Antilatency Device Network Alt

Nodes

При подключении к сети каждое устройство получает идентификатор NodeHandle. Идентификаторы не повторяются в процессе работы одного экземпляра сети.
